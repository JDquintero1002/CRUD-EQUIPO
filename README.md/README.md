# 🛠️ Sistema de Gestión de Equipos - API REST

Este proyecto es una API REST desarrollada con **Django** y **Django Rest Framework (DRF)** para llevar el control del ciclo de vida (CRUD) de los equipos de mantenimiento técnico de hardware.

## 🚀 Requisitos e Instalación

Sigue estos pasos para clonar y ejecutar el proyecto localmente:

### 1. Clonar el proyecto
```bash
git clone [https://github.com/JDQuintero1002/CRUD-EQUIPOS.git](https://github.com/JDQuintero1002/CRUD-EQUIPOS.git)
